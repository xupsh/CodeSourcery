VL Verify Load

Availability: ALL

Syntax

VL [ [-[N]O scn_types] filename ... ] ...

Description

The Verify Load command is used to verify the program was downloaded correctly.

The Verify Load command is used to verify a program load. When no arguments are specified, all sections of all files previously downloaded are uploaded and checked against the original executable files.

  • scn_types - A set of letters specifying section types to verify (-O) or not verify (-NO):

t text (program code)

d data (initialized data)

b bss (uninitialized data)

l literals (read-only initialized data)

  • filename - If specified, the executable file filename is uploaded from the target and verified against the original program file. Otherwise, the previously loaded file(s) is verified. The sections to be verified are determined by scn_types.
