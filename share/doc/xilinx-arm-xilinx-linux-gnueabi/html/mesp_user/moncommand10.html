CF, CFI, CI Cache Flush

Availability: ALL

Syntax

CF [I|D]

CI [I|D]

CFI [I|D]

Description

The Cache commands are used to flush and/or invalidate the contents of the instruction and/or data caches.

  • CF - flushes (writes back to memory).
  • CI - invalidates without updating memory.
  • CFI - flushes and then invalidates.

If an I or a D operand is specified, only the instruction (I) or data (D) cache is affected. If no operand is specified, both caches are affected.

Note

Support for Cache Flush (CF) and Cache Invalidate (CI) operations depends on features that are not provided by all processors. If the specific processor you are using does not support the operation, an error message is displayed.
