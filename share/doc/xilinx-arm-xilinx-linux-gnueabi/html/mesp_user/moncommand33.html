KN Kill Names

Availability: ALL

Syntax

KN {*|ident|ident*}

Description

The Kill Names command deletes one or more symbols from MON’s symbol table.

  • * - Kills all symbols. ‘*’ is a wildcard character and can be given alone or at the end of ident. When used with ident all matching names are deleted.
  • ident - Is a global data, function, or user-defined (entered with EN) symbol name. ident is case sensitive. When used with ‘*’, ident can be the starting characters of one or more global data, functions, or user-defined symbols.

See also LN Load Names and EN Enter Names.
