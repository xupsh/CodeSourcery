Supplied Components

Before you use the Sourcery Probe Professional, verify that these components are present:

  • The external power supply with US cord and adapters (3) EUR, UK, AUST.
  • One ADAP, MOD JCK/RS232, Female (9pin D).
  • One ADAP, MODJCK/RS232, Male (9pin D).
  • One Debug Probe Tip.
  • One cable Ethernet, straight, 6ft.
  • One cable, USB 2.0 A-M to B-M, 1.2 m.
  • One RJ11 cable, 7ft.
  • One flexible probe tip extension cable.
  • One cable, assy, 50 pin, flat ribbon 18 in. (Standard debug port probe tip cable assembly).
