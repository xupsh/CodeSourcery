Sourcery Probe Personal Console Menu

The Sourcery Probe Personal Console Menu (Figure 2‑3) provides you with several options including:

  • Choose Replay boot log to view the probe configuration information.
  • Choose USB/Net settings to set local link mode, or to enter a static IP address.
  • Set a recognizable name for the unit, which will be displayed by the probe discovery feature of the CodeBench debugger.

You can access this menu once you have established a connection between your probe and a terminal emulator by pressing the <ESC> key.
