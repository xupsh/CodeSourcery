Mentor Embedded Sourcery Probe Personal (Sourcery Probe Personal) is a cost-effective tool that helps you develop and debug a number of processors and microcontrollers.

Tip: If you have a probe that is labeled Mentor Embedded USB-JTAG Probe, this works exactly the same as the Sourcery Probe Personal. The only difference is the label on the hardware. Follow the instructions in this manual for Sourcery Probe Personal.
