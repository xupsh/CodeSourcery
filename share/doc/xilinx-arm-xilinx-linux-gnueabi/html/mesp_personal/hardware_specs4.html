Debug Port Connector

The debug connector consists of a 5.5 inch ribbon cable with the appropriate debug connector attached. The ribbon cable has a red stripe down one side to indicate the location of pin 1.

Note

There are many different standard debug connectors. Target Adapter Modules are available to support alternate debug connector types. It is very important to check the debug connector details of your target board when selecting the right adapter module. See the Mentor Embedded Sourcery Probe Target Connection application note and Table 2‑1.
